Sistema de Reportería para ERP

Contexto de Proyecto

Como parte del desarrollo de un ERP universitario, me encargué de implementar el sistema de reportería personalizado. Este módulo estaba diseñado para generar informes visuales utilizando HTML, CSS y JavaScript. Para conectar con los datos requeridos, diseñé e implementé múltiples APIs específicas, una por cada tipo de reporte.

Aporte realizado

Durante el proyecto, aprendí a implementar Google Charts para la creación de gráficos interactivos y eficientes, logrando una presentación visual clara de los datos. Además, diseñé e implementé las APIs necesarias para conectar los datos del sistema, asegurando una experiencia fluida y optimizada para los usuarios.

Sistema de Hotelería de Animales con Microchips

Contexto de Proyecto

Este proyecto universitario se desarrolló en equipo, y su objetivo era gestionar animales mediante el uso de microchips. Se utilizó Bootstrap para el diseño del front-end y C# para el back-end, construyendo un sistema práctico y moderno.

Aporte realizado

Mi rol en el equipo se centró en los mantenimientos generales del sistema, abordando tanto ajustes en el código del front-end como en el back-end. Este proyecto me permitió trabajar de manera efectiva en colaboración con mis compañeros, asegurando que todas las funcionalidades estuvieran correctamente integradas y funcionando según lo planeado.

Sistema de Prevención de Demencia

Contexto de Proyecto

Proyecto enfocado en el desarrollo de una web destinada a la prevención de la demencia, proporcionando herramientas para apoyar a los usuarios en el cuidado de su salud cognitiva.

Aporte realizado

Mi contribución principal fue el desarrollo de dos funcionalidades clave: un foro interactivo donde los usuarios podían comentar y comunicarse entre sí, y un mini-juego de rompecabezas diseñado para estimular la cognición. Este proyecto me permitió aplicar mis habilidades tanto en el diseño como en la programación de funcionalidades específicas que agregaron valor al sistema en su conjunto.

Mejora de Eficiencia en el Sistema de Boletas

Contexto de Proyecto

istema corporativo para la gestión de boletas de trabajo que incluía funciones como bitácoras, comentarios, contactos y envío de correos. Aunque funcional, el sistema presentaba problemas de organización y eficiencia.

Aporte realizado

Diseñé e implementé un sistema de backlog asociado a cada boleta, lo que mejoró significativamente el flujo de trabajo y la priorización de tareas. Además, optimicé varios procesos internos, reduciendo el tiempo de respuesta y aumentando la satisfacción de los usuarios. Estas mejoras reflejaron una comprensión profunda de las necesidades del sistema y de los usuarios.

Migración Funcional y Estética de Pantalla (Asientos Contables)

Contexto de Proyecto

Pantalla utilizada para generar asientos contables en el módulo de cuentas por cobrar. El diseño anterior carecía de modernidad y no facilitaba una experiencia fluida para el usuario.

Aporte realizado

Lideré la migración a un diseño renovado que priorizó tanto la funcionalidad como la estética. Implementé componentes visuales modernos y mejoré la organización de los elementos para facilitar su uso. Esto no solo incrementó la eficiencia de las tareas realizadas, sino que también mejoró notablemente la experiencia de los usuarios al interactuar con el módulo.
